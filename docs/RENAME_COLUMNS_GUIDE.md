# Renomear Colunas do Board CSM

## URL do Board
```
https://whd.atlassian.net/jira/core/projects/CSM/board?filter=&groupBy=status
```

## Passos para Renomear

### 1. Acessar Board Settings
- No board, clique no **ícone de engrenagem** (⚙️) no canto superior direito
- Selecione **Board settings**

### 2. Ir para a aba "Columns"
- Na janela de settings, abra a aba **Columns**
- Você verá as colunas atuais (ex: "To Do", "In Progress", "Done")

### 3. Renomear Cada Coluna

**Encontre cada coluna e renomeie:**

| Coluna Atual | Novo Nome | Etapa |
|---|---|---|
| Coluna 1 | Análise de Perfil | 1 |
| Coluna 2 | Implantação | 2 |
| Coluna 3 | Lançamento | 3 |
| Coluna 4 | Acompanhamento 1 | 4 |
| Coluna 5 | Acompanhamento 2 | 5 |
| Coluna 6 | Expansão | 6 |
| Coluna 7 | Renovação | 7 |
| Coluna 8 | Cancelamento | 8 |

### 4. Adicionar Colunas Faltantes (se necessário)
Se houver menos de 8 colunas, clique em **+ Add column** para adicionar as que faltam.

### 5. Salvar
Clique em **Save** ou **Done**

## Dúvidas?

Se não conseguir renomear via UI, o board pode estar em um estado especial. Tente:
- Recarregar a página (F5)
- Acessar Board Settings novamente
- Ou criar um novo board manualmente via "Create board"
