# ✓ Checklist Operacional Diário/Semanal

Rotina prática para usar o Jira CS no dia a dia.

---

## **ROTINA DIÁRIA** ⏰

### **INÍCIO DO EXPEDIENTE (09:00 - 10:00)**

#### **1. Revisar o Board (10 min)**
- [ ] Entro em **Board**
- [ ] Vejo todos os clientes
- [ ] Identifique clientes que devem mudar de coluna
  - Cliente saiu do Onboarding? Move para Ativo
  - Cliente vai renovar? Move para Renovação
  - Cliente churned? Move para Cancelamento
- [ ] Crio plano do dia

#### **2. Revisar Dashboards (5 min)**
- [ ] Abro **Dashboard Saúde da Base**
- [ ] Vejo se há clientes em risco
- [ ] Se sim: Agendar reunião hoje se possível
- [ ] Reviso clientes sem contato há 60d

#### **3. Verificar Emails/Mensagens (5 min)**
- [ ] Email de cliente? Crie **Interação** registrando contato
- [ ] Problema relatado? Crie **Risco** se relevante
- [ ] Oportunidade mencionada? Crie **Oportunidade**

#### **4. Revisar Seus Tickets (5 min)**
- [ ] Vejo tarefas do dia anterior que ficaram pendentes
- [ ] Priorizo: Reuniões > Riscos > Oportunidades > Rotina

---

### **DURANTE O DIA**

#### **Reunião com Cliente?**
DEPOIS da reunião (mesmo dia):
- [ ] Crie **Interação** registrando a reunião
  - Tipo: Reunião
  - Data: Hoje
  - Resumo: O que foi discutido
  - Próximas etapas: O que vai fazer
- [ ] Se descobre risco: Crie **Risco**
- [ ] Se cliente mencionou expansão: Crie **Oportunidade**
- [ ] Atualize **Ciclo da Conta** se mudou

#### **Cliente Fez Contato?**
- [ ] Crie **Interação** registrando
  - Tipo: Suporte / Feedback / Outra
  - Data: Hoje
  - Resumo: O que cliente falou
  - Ação: O que vai fazer

#### **Descobriu um Problema?**
- [ ] Crie **Risco** imediatamente
  - Descrição clara
  - Probabilidade e impacto
  - Plano de ação

#### **Oportunidade de Expansão?**
- [ ] Crie **Oportunidade** registrando
  - O que é a oportunidade
  - Valor estimado
  - Próximas etapas

---

### **FIM DO EXPEDIENTE (17:00 - 17:30)**

#### **Revisar o Dia**
- [ ] Todas as reuniões foram registradas como **Interações**?
- [ ] Todos os riscos foram documentados?
- [ ] Todas as oportunidades foram criadas?
- [ ] Board atualizado com movimentações?

#### **Preparar Próximo Dia**
- [ ] Clientes em risco? Agendar reunião amanhã
- [ ] Clientes sem contato? Marcar call
- [ ] Renovações próximas? Verificar status

## **ROTINA SEMANAL** 📅

### **SEGUNDA: Saúde da Base** 🏥

**Manhã (30 min)**
- [ ] Abro **Dashboard Saúde da Base**
- [ ] Vejo:
  - Quantos em risco?
  - Quantos Health Score baixo?
  - Quantos sem contato 60d?
- [ ] Para cada cliente com problema:
  - [ ] Agendar reunião esta semana
  - [ ] Definir plano de ação
  - [ ] Marcar no calendário

**Ação esperada:**
```
Se 3 em risco:
- Reunião hoje com 1
- Reunião quarta com 1
- Reunião sexta com 1
```

---

### **TERÇA: Seguimento** 🔄

**Manhã (15 min)**
- [ ] Reviso clientes em **Acompanhamento 1**
- [ ] Health Score de cada um
- [ ] Ficam de contactar? Contactar hoje
- [ ] Próximas etapas no plano de sucesso?

**Ação esperada:**
```
Clientes em Acompanhamento 1 precisam de
acompanhamento mais próximo:
- Responder rápido
- Oferecer suporte
- Celebrar milestone
```

---

### **QUARTA: Relacionamento** 👥

**Meio da tarde (20 min)**
- [ ] Abro **Dashboard Relacionamento**
- [ ] Vejo:
  - Quantas interações registrei até agora (meta: 40+)?
  - Tipos de interação: reuniões, suporte, feedback?
  - Participação em eventos?
- [ ] Se interações baixas:
  - [ ] Agendar mais reuniões para esta semana
  - [ ] Convidar clientes para evento

**Ação esperada:**
```
Se 15 interações (meta 40):
- Faltam 25 interações
- Agendar 5 reuniões (5 interações)
- Responder 10 suportes (10 interações)
- Acompanhar 5 clientes (5 interações)
- Evento: 5 clientes participando
```

---

### **QUINTA: Oportunidades & Renovações** 💰

**Manhã (20 min)**
- [ ] Abro **Dashboard Crescimento**
- [ ] Vejo:
  - Oportunidades abertas? Status de cada uma?
  - Renovações próximas (90 dias)? Começou negociação?
  - Indicações? Acompanhando?
- [ ] Para cada oportunidade:
  - [ ] Próximo passo é...?
  - [ ] Quando vai fazer?
- [ ] Para cada renovação:
  - [ ] Reunião agendada? Proposta enviada?

**Ação esperada:**
```
Se 3 oportunidades:
1. Oportunidade A: Fazer demo (quinta)
2. Oportunidade B: Enviar proposta (sexta)
3. Oportunidade C: Call com finance (próxima semana)

Se 5 renovações:
1. Reunião com cliente 1 (segunda)
2. Reunião com cliente 2 (quarta)
3. Proposta para cliente 3 (quinta)
4. Reunião final cliente 4 (segunda próxima)
5. Assinatura cliente 5 (prazo sexta)
```

---

## **ROTINA MENSAL** 📈

### **INÍCIO DO MÊS (1-2 dias)**

**Revisar & Atualizar Ciclos**
- [ ] Abro **Filtro** para listar todos clientes
- [ ] Reviso **Ciclo da Conta** de cada cliente
  - [ ] Cliente começou novo trimestre? Atualize Ciclo
  - [ ] Cliente é novo? Marque como Onboarding
  - [ ] Cliente em risco? Marque como Risco
  - [ ] Cliente renovando? Marque como Renovação

**Atualizar Board**
- [ ] Clientes que saíram de Análise? Mover para próxima coluna
- [ ] Clientes prontos para Cancelamento? Mover
- [ ] Revisar cada coluna para mover quem tem de mover

---

### **MEIO DO MÊS**

**Reunião de Alinhamento**
- [ ] Consolidar dashboards
- [ ] Apresentar:
  - Carteira: total, em risco, saudável
  - Engajamento: interações, participação
  - Pipeline: oportunidades, renovações
  - Churn: nenhum / 1 / X

**Decisões**
- [ ] Priorizar clientes em risco
- [ ] Oferecer suporte extra para quem precisa
- [ ] Agendar eventos / atividades

---

## **Checklist: Rotina Implementada?**

- [ ] Consulto Board todos os dias
- [ ] Consulto Dashboards 3x por semana
- [ ] Crio Interação após cada reunião
- [ ] Crio Risco quando identifico problema
- [ ] Crio Oportunidade quando há chance
- [ ] Atualize Ciclo da Conta quando muda
- [ ] Movo clientes no Board quando progride
- [ ] Faço relatório semanal
- [ ] Consolido métricas mensais
- [ ] Compartilho com time regularmente

**Quando conseguir fazer tudo, sua operação será excelente! 🎯**
