# 📝 Exemplo Prático Completo: ACME Corp

Um exemplo realista e preenchido de como usar a estrutura Jira CS de ponta a ponta.

---

## **Contexto do Exemplo**

Vou contar a história de **ACME Corp**, uma empresa de tecnologia que assinou contrato conosco em **janeiro de 2026**.

**Dados básicos:**
- Empresa de software, 150 usuários
- Segmento: Enterprise
- Contrato: R$ 50.000/mês
- Início: 01/01/2026
- Renovação prevista: 30/06/2026

## **Lições Deste Exemplo**

✅ **Cliente** = um registro único por empresa, atualizado conforme evolui

✅ **Interações** = registro de cada toque importante (reunião, feedback)

✅ **Plano** = estrutura para os 90 dias iniciais, guia operacional

✅ **Oportunidades** = tanto para receita quanto para marketing

✅ **Renovação** = começa 90 dias antes, acompanha todo processo

✅ **Board + Ciclo da Conta** = dois layers complementares, não iguais

✅ **Tudo vinculado** = nada fica solto, tudo conecta ao Cliente

---

## **Como Replicar Este Exemplo**

Se você tem um cliente como ACME:

1. **Dia 1:** Cria Cliente (nome, dados básicos)
2. **Dia 1:** Cria Interação de kickoff
3. **Dia 1:** Cria Plano de Sucesso (90 dias)
4. **Semanal:** Cria Interações
5. **Conforme aparecer:** Cria Risco, Oportunidade
6. **90 dias antes:** Cria Renovação
7. **Sempre:** Atualiza Ciclo da Conta conforme situação muda

**Pronto! Você tem documentação viva da relação com o cliente. 📊**
