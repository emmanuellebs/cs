# Criar Board WH - Customer Success (Manual)

## Passos Rápidos

1. **Acesse o Jira**: https://whd.atlassian.net

2. **Navegue para o projeto CSM**
   - Clique em **Projects** > **CSM**

3. **Criar novo board**
   - No menu lateral, procure por **Boards** (ou procure "Create board")
   - Clique em **Create board** ou **+ Create a new board**

4. **Escolher tipo**
   - Selecione **Kanban board** (não Scrum)

5. **Configurar board**
   - **Board name**: `WH - Customer Success`
   - **Project**: Selecione **CSM**
   - Clique **Create**

6. **Renomear colunas** (após criação)
   - Clique no ícone de engrenagem (⚙️) > **Board settings**
   - Vá para **Columns**
   - Renomeie para:
     1. Análise de Perfil
     2. Implantação
     3. Lançamento
     4. Acompanhamento 1
     5. Acompanhamento 2
     6. Expansão
     7. Renovação
     8. Cancelamento
   - Clique **Save**
